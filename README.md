### :dart: Features
- Add Book
- Browse Book
- Live Chat By using Tawk.to 
- Session Storage By jsonWebToken
- User Authentication
- Track Order
- Manage Order
- Sweetalert-2 for User Interaction
- PocoLoco animation
- FontAwesome icons
- Validation on Forms using YUP

  
### :bangbang: Technologies Used

- React
- MongoDB
- Node.js
- Express
- Postman API
- HTML
- CSS
- Javascript
- Bootstrap
- This project uses npm as package manager

## :toolbox: Getting Started

### :running: Run Locally

Go to the frontend directory
```bash
cd frontend
```
Install dependencies
```bash
npm install
```
Start the server
```bash
npm start
```
Go to the backend directory
```bash
cd backend
```
Install dependencies
```bash
npm install
```
connect backend
```bash
npm run dev
```


### :triangular_flag_on_post: Deployment

To deploy this project manually run
```bash
npm run build
```
